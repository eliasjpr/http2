# http2

HTTP/2 implementation in [Crystal](http://crystal-lang.org/). Heavily inspired
by [igrigorik/http-2](https://github.com/igrigorik/http-2).

Not usable yet.

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  http2:
    github: markauskas/http2
```

